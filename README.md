## ğŸ§  Gameplay Loop

1. The robot runs automatically forward.
2. Shift left/right to align with platforms.
3. Jump gaps between buildings.
4. Collect **Scrap** to increase score and progression.
5. As time passes, the game **speeds up** and becomes more difficult.
